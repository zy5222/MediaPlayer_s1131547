# MediaPlayer_s1131547

##  核心功能介紹

* **1. 瀏覽影片 (選擇檔案)**：
  點擊「瀏覽」按鈕後，會開啟檔案選擇視窗，讓使用者從電腦中挑選想要觀看的影片檔案。
* **2. 開始播放**：
  載入影片後，點擊「播放」按鈕，即可開始欣賞影片內容。
* **3. 暫停播放**：
  在觀看影片的過程中，若需要稍微離開，可以隨時點擊「暫停」按鈕，畫面會停留在當下的影格。
* **4. 停止播放**：
  點擊「停止」按鈕，影片會完全中斷並回到最一開始的狀態，方便使用者重新播放或是準備切換下一部影片。

##  防呆機制與設計

**1. 專屬影片格式過濾：**
為了幫助使用者快速找到影片檔，點擊「瀏覽」開啟視窗時，系統會自動幫忙過濾並只顯示常見的影片格式（如 `.mp4`, `.wmv`, `.avi`）。這樣就避免不小心點到文件或圖片檔導致無法播放。

**2. 載入檔案不自動播放 (防突發音量)：**
在挑選並載入影片後，程式會先保持「停止」的安靜狀態，等待使用者準備好並主動按下「播放」按鈕才會開始。這個設計可以避免使用者在安靜的環境中，因為一選好檔案就突然發出巨大聲音。
